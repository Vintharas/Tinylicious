Tinylicious: A Simply Delicious jQuery Widget

MyDelicious

Tinylicious?

Tinylicious is a very simple delicious widget coded using jQuery and JSON.

Current features are:

  • Retrieval and display of the most recent delicious public bookmarks of a given user.

Setup!

The setup is pretty straightforward:

1. Add reference to jQuery libraries: 

<head>
 <!-- head stuff -->
 <script type="text/javascript" src="lib/jquery/jquery-1.4.2.min.js">
 </script>
</head>

2. Add reference to Tinylicious javascript file: 

<body>
 <!-- all the contents -->
 <div id="scripts-at-the-end">
  <script type="text/javascript" src="tinylicious.js"></script>
 </div>
</body>

3. Add this html code wherever you want to put the widget: 

<div id="tinylicious">
 <p class="title"><span class="my">My</span>Delicious</p>
 <ol>
 </ol>
</div>

4. Add the stylesheet for the widget or make your own: 

<head>
 <link type="text/css" rel="stylesheet" href="tinylicious.css" />
</head>

5. And finally add this script: 

<script type="text/javascript">
 $(document).ready(function(){
  Tinylicious.LoadUserBookmarks('username', numberOfBookmarks);
 });
</script>

where username will be most likely your username in delicious and numberOfBookmarks the number of bookmarks you want the widget to display.
